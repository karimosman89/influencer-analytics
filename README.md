# Social Media Influencer Analytics
This project analyzes social media influencers' performance metrics.

## Overview
The analysis evaluates engagement rates and audience growth over time.

## Data Source
- Scraped social media data or use a public dataset.

## Setup
Install dependencies:


      pip install -r requirements.txt


## Usage
- Run the scraper:


       python src/scraper.py

  
- Analyze data:


       python src/analysis.py

  
- Generate visualizations:



      python src/visualization.py
